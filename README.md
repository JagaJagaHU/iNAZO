# HUsearch

HUsearchとは北大が公開している成績データを見やすく纏めたWebアプリケーションです。

過去の成績は講義を選択する時の指標の１つとして参考になります。

# Features

- グラフで表示
- 検索
- ソート機能
- 講義のブックマーク登録
